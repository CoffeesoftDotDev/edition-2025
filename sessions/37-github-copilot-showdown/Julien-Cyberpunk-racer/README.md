### Code Structure

- **`main.py`**: Initializes Pygame and runs the main game loop
- **`game_core.py`**: Contains central game logic (collision, scoring, states)
- **`graphics_manager.py`**: Manages sprite rendering and display
- **`sound_manager.py`**: Manages sound effects and music
- **`game_inputs_manager.py`**: Captures and processes keyboard/mouse inputs
- **`entities/`**: Classes for all game objects (player, enemies, items, etc.)
